## 模块一：基础与准备
1. Bash脚本: Bash（Bourne Again SHell）是Linux系统中常用的命令解释器和脚本编程语言。脚本文件本质上是包含一系列命令的文本文件，通过解释执行实现自动化任务。
2. 创建第一个脚本:
    - 创建sh后缀的脚本文件;
    - Shebang行: "#!"脚本文件的第一行，告诉系统脚本文件的解释器;
    - 添加执行权限;
    - 运行脚本: 
        - sh script.sh;
        - bash script.sh;
        - chmod +x script.sh & ./script.sh;
        - source script.sh;
        - . script.sh;
    - 备注:
        source(.) 命令不创建子shell进程, 用于在当前shell环境读取并执行文件内的命令, 返回最后一条命令的执行结果;
3. 基本命令和快捷键:
    - 常用基础命令:
        - echo "输出文本";
        - pwd "显示当前目录";
        - ls "列出目录内容";
        - cd "切换目录";
        - cat "显示文件内容";
        - date "显示日期时间";
    -终端快捷键:
        - Ctrl + C: 中断当前正在执行的命令;
        - Ctrl + D: 退出当前终端;
        - Ctrl + L: 清屏;
## 模块二：核心编程语法
1. 变量:
    - 变量的定义和使用:
        - 定义变量(等号两边不能有空格);
        - 加$符号使用变量;
        - 使用{}明确变量边界;
        - 变量命名规则: 只能包含字母, 数字和下划线, 不能以数字开头, 区分大小写;
    - 位置参数:
        - 脚本名称: $0;
        - 第一个参数: $1;
        - 第二个参数: $2;
        - 所有参数个数: $#;
        - 所有参数列表: $@;
        - 所有参数字符串: $*;
        - $@和$*的区别:
            "$@" 和 "$*" 都会返回所有参数列表, 但是 "$@" 返回参数列表中的每个参数, 而 "$*" 返回参数列表中的所有参数作为一个字符串;
    - 特殊变量:
        - $?: 上一条命令的退出状态码(0表示成功);
        - $$: 当前shell的进程pid;
        - $!: 最后一个后台进程的pid;
        - $RANDOM: 随机数(0-32767);
2. 引号的使用:
    - 双引号: 允许变量扩展;
    - 单引号: 不允许变量扩展, 原样输出;
    - 反引号: 允许命令替换(执行命令并返回结果);
        - 现代写法: 用$(command)替换`command`;
    - 转义字符;
3. 算术运算:
    - 算数扩展语法: $((expression));
    - let命令;
4. 条件判断:
    - test命令和[]语法: 
        - 用于评估表达式并返回布尔值;
        - 格式: test expression 或 [ expression ];
    - 比较运算符:
        - 数值比较:
            -eq(equal);
            -ne(not equal);
            -gt(greater than);
            -lt(less than);
            -ge(greater than or equal)
            -le(less than or equal);
        - 字符串比较:
            =或==(字符串相等);
            !=(字符串不相等);
            -z(字符串为空, 长度为0);
            -n(字符串非空);
        - 文件测试:
            -f(是否为普通文件);
            -d(是否为目录);
            -e(文件是否存在);
            -r(文件是否可读);
            -w(文件是否可写);
            -x(文件是否可执行);
